# Latent Art Canvas V3

这是完整可运行版本。

## 文件

- `index.html`：双层解码 + 横纵解耦 + 白缝修复 + Latent Map + Decoder View 的完整单文件 HTML。

## 使用

直接用浏览器打开 `index.html` 即可。

## 在线展示

这个项目是纯静态页面，可以直接部署到 GitHub Pages。上传到 GitHub 后，在仓库的 `Settings -> Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`，保存后等待部署完成。

## 主要变化

- 修复临近同色之间出现白色间隙的问题：
  - 条纹位置和宽度改为像素对齐整数。
  - 主 Canvas 设置 `imageSmoothingEnabled = false`。
  - 绘制时使用整数 `fillRect` 边界。

- 去掉底部条码作为主要可视化，新增：
  - `LATENT MAP`：当前图像在高维隐空间中的 2D 投影点。
  - 轨迹线：连续演化序列，并用科学色图映射每步距离/速度。
  - `DECODER VIEW`：用紫到黄的科学色图小格展示 `Z -> H1 -> H2 -> OUT`。
- 降低序列跳变：
  - 条纹抽样改为 universe seed 派生的稳定采样。
  - 降低 latent 演化步长。
  - 降低整体条纹密度，接近 V2 的留白和块面节奏。
  - 新增 `RANDOM` 控制，允许在连续演化和更强跳变之间调节；`0` 会冻结演化。
  - 大变化节奏由 decoder 输出的随机 tempo/phase 参数决定，而不是固定外部节拍。
- 新增控制项：
  - 颜色数量滑杆：`AUTO` 由 decoder 决定；数字模式强制使用对应数量。
  - 色板手选模式：未手选时显示当前激活色，点选后只使用手动颜色，`AUTO` 清空手选。
  - 色板位于 `DECODER VIEW` 下方，采用 4x8 排列。
  - 参数表移动到左侧色板下方。
  - `XY` / `X` / `Y` 方向模式。
  - 中英文切换。
  - 展览模式、PNG 导出、状态持久化。
  - 点击 `LATENT MAP` 可跳转到附近隐空间区域。

- 保持：
  - 横向/竖向条纹完全不透明遮盖。
  - 双隐藏层解码器。
  - 横向、竖向、全局、色彩参数解耦。
  - 无颗粒噪点绘制。
